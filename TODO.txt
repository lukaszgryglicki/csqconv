To access cluster: you need to install `awscli` on your OS.

Then `mkdir ~/.aws/`
`vim ~/.aws/config`:
```
[profile cncf]
region = us-east-1
output = json
```

`vim ~/.aws/credentials`:
```

aws_access_key_id = AKIAU4VUTP77BILEVAOJ
```
